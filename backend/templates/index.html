Network Packet Sniffer API

This is the API for the Network Packet Sniffer developed using Python and Flask.

GET /packets

Retrieve a list of captured network packets.

GET /system

Retrieve system performance data, including CPU usage.

POST /scan

Perform a port scan on a target IP address.

Developer Information

Developed by Anil Sinthu, this API aims to provide real-time insights into network activity and system performance.

API Response

API responses are in JSON format, providing detailed information about network packets, system performance, and port scan results.
